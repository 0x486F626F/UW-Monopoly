Building Buyer 7000
===

Compilation:
---

###Environment Requirement:
g++ complier 4.2 or above
X environment (graphic display only)

To have only text display:
	1. cp textMakefile Makefile
	2. make

To have graphic display:
	1. cp graphicMakefile Makefile
	2. make
